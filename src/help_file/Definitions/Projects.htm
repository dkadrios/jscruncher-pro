Projects

Projects are meant to help out with repetitive tasks.  A Project contains:

  • Location of files to compress.

  • Location of files to manage script-includes.

  • Compression and obfuscation settings.

 

Projects are stored in the [program dir]\projects folder in an XML format.  You can either manually edit them, or create and edit them through the JSCruncher UI, on the Home page.

 

Projects can be run by first loading them from the Home page, and then selecting 'run'.  Projects can also be run from the command line tool, making them ideal for use in build scripts.
