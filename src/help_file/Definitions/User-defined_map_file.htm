User-defined map file

JSCruncher maintains a special map file which is referred to as the "user-defined" map file.  Items get added to this map when you right-click on them from the session map pane and select "Add to user defined map", causing them to be ignored by the obfuscation engine.

 

To remove items from the user-defined map, you can either manually edit the file, or use the controls in the user defined pane.  

The user-defined map file is located in the map folder and is named "user_defined.jmap" by default.  You can alter this name on the environment settings page.
