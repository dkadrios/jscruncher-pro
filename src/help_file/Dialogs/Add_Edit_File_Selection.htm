Add/Edit File Selection

This dialog is used for adding or editing file selections from the project editing screen.

 

  • Input Folder

The folder that the files reside in.

  • Output Folder

The destination folder.  If this is the same as the input folder, the application will raise warnings before performing operations.

  • File/Mask

A specific file or generic mask of files to process.  Accepts standard Windows wild-cards (i.e. *.js)  You can define multiple masks by separating them with semicolons.

  • Include Subdirectories

Whether or not to recursively process sub-directories.

  • Filename prefix

Optional characters to be prepended to each filename.

  • Filename suffix

Optional characters to be appended to each filename.
