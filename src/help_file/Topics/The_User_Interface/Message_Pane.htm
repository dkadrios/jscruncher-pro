Message Pane

If the program encounters any errors while validating or compressing files, it will report them in the message pane.  This pane appears in the lower portion of the screen.  Note that it only appears when it has content.

Depending on the nature of the message, you can right-click the message to invoke the built-in file editor, to correct the error condition.  For instance, one common message is 'possible missing semi-colon'.  This is shown when the validator determines there to be a missing semi-colon.  You can double-click the message to bring up the file editor and correct it.  The file editor will automatically move the cursor to the area with the missing semi-colon.
