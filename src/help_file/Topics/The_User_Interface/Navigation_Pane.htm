Navigation Pane

Dominating the left-hand side of the application is the Navigation Pane.  This is used to get at all the various tasks that JSCruncher Pro can perform.

The available pages are listed as buttons at the bottom of the page.  The content of the currently selected pane is shown in the top portion.

 

The available pages are:

 

  • Home

  • Compress Files

  • Compress Text

  • Manage Tags
