Excluding tokens

You can quickly exclude tokens from being obfuscated by right-clicking on them when they appear in the lower session map pane and selecting "Add to user defined map".  This will place entry in the "user-defined map file" causing the token to be ignored by the obfuscation engine.  You can also add and remove tokens from the user-defined map file using the controls in the User Defined Mapfile pane.

 

See map files for more information on tokens.
