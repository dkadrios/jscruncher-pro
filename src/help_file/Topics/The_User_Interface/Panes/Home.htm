The Home Page

The home page is displayed when you first load the program.  You can return to it at any time by click on the 'Home' button in the left-hand navigation pane.

 

From the home page, you can perform project related tasks:

  • Create a new project

  • Load an existing project

 

  • Run current project

  • Edit current project

  • Save current project as...

 

You can also jump from the home page to other features of the program:

  • Compress Files

  • Compress Text

  • Manage Tags
