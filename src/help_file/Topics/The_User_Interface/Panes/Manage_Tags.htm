Manage Tags

This pane allows you access to the tag management routines of the program.

The left hand side provides a tree from which you can choose the folder containing your web pages.  On the right is a file listing from which to select files.  You can double-click any file to invoke the built-in editor.

 

The controls above the file listing include a 'mask filter' control and a checkbox to 'include sub-directories'.   Once all the files you want to manage are showing, select them and press either the 'alter to use compressed scripts' or 'alter to use normal scripts' to manage your script tags..  

 

At the bottom of the screen you'll notice several information panes.  These include the 'file viewer' which will show the contents of a selected file, and a 'script includes' pane which lists all the scripts tags found in the selected file.
