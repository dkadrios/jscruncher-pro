Settings

The settings screen is divided into 2 tabs:

 

Obfuscation

Confirmations/Popups

 

 

Obfuscation

  • Location of map files

This is the path that JSCruncher will search when looking for obfuscation map files.  This can be a local folder, a network share or UNC.

  • User-defined map

Specifies which map in the folder is to be used as the user-defined mapfile.  

In the session map that appears at the bottom of the screen after obfuscation, are all the substitutions that JSCruncher made.  If you right-click any of these and select "Never replace", they are placed into what is known as the "user defined" map, which resides in the map folder with the other maps.

 

 

Confirmations/Popups

Listed here are checkboxes for all the 'confirmations' dialogs that JSCruncher may raise.  You can turn them on or off.

There also is a checkbox for controlling whether or not the compression settings dialog is displayed before each run.
