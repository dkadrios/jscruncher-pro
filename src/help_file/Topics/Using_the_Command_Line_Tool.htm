Using the Command Line Tool

JSCruncher Pro includes a command line version .  This makes it easy to integrate JSCruncher into your build scripts.

 

The command line tool is called jscrunch32.exe and is located in your installation folder.  

 

Usage

Typing jscrunch32 /? at a command prompt (while in the folder you installed to) will give you a listing of the available syntax that the command line supports:

 

-------------------------------------------------------------

-------------------------------------------------------------

Usage: jscrunch32.exe [project]

 Example:

 jscrunch32 c:\jscruncher pro\projects\myProject.jscp

-------------------------------------------------------------

Examples

Run the project file 'c:\jscruncher pro\projects\myProject.jscp':

jscrunch32 c:\jscruncher pro\projects\myProject.jscp
